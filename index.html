Over the years,artificial intelligence and machine learning has been used to solve a lot of problems that has been unsolved for years.The concept of AI involves a computer to think like human beings while machine learning which is a subset of AI is the process of teaching a computer to learn from previous experience(input data) using an algorithm,adapt(new input data) and provide solutions i.e output. I've been able to solve a classification problem on an image dataset.Classification is a type of supervised learning which involves training a network to learn to classify data i.e put them in the box they belong to.These boxes are called classes.Take for instance,the spam box in our email. The problem of spam mails was solved using classification.The email system is able to distinguish between a legitimate and an illegitimate email based on the training it's recieved and also from the properties of the email.Properties like source of the email and so on .

This classification problem was performed on an image dataset called CIFAR-10.This dataset contains 10 classes of images,each class contains 10,000 images which brings the total number of images to 60,000 but because of GPU limitations and also because training time ,i reduced the number of classes to 5 with each class containing train 1000 images which brings the total number of train data to 5000 and the test data to 1000.To solve this problem,i made use of a deep learning API called keras.
According to wikipedia,Keras is an open source neural network library written in Python. It is capable of running on top of TensorFlow, Microsoft Cognitive Toolkit, or Theano. Designed to enable fast experimentation with deep neural networks, it focuses on being user-friendly, modular, and extensible. Tensorflow is an open-source platform for machine learning.It provides workflow with high level APIs.I made use of the sequential model and VGG network architecture. The VGG model is made up of alot of layers but i made use of the following;

  • Convolutional Layer
  • Pooling layer
  • MaxPooling layer
  • Flatten layer
  • DropOut layer
  • Dense layer

After the training and validation was carried out ,the network was able to achieve an accuracy of 81.3% which is a good start because if i begin to add some optimization techniques like adding more layers and sublayers to the network or increasing the epoch size, it'll be able to achieve higher accuracy.Below is the loss and accuracy curve

Loss and Accuracy Curve (Blue curve:Accuracy,orange curve:Loss) As you can see from the curve,there's no overfitting in the network as the accuracy and loss curve are close to each other.
